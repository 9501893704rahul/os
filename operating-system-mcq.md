# Operating System - Lesson 1: MCQ Practice Questions

## Multiple Choice Questions with Answers

---

### Section A: Basic Concepts of Operating System

#### Q1. What is an Operating System?
- (a) A hardware component
- (b) A set of programs that control the execution of application programs ✅
- (c) An input device
- (d) A storage device

**Answer: (b)**
> Explanation: An Operating System is a set of programs that control the execution of application programs and act as an intermediary between user and computer hardware.

---

#### Q2. Operating System is an example of:
- (a) Application Software
- (b) Utility Software
- (c) System Software ✅
- (d) Firmware

**Answer: (c)**
> Explanation: Operating System is technically an example of System Software because it directly interacts with hardware.

---

#### Q3. Which of the following is NOT a function of Operating System?
- (a) Resource Management
- (b) Memory Management
- (c) Creating documents ✅
- (d) Process Management

**Answer: (c)**
> Explanation: Creating documents is done by application software (like MS Word), not by the Operating System.

---

#### Q4. Operating System acts as an intermediary between:
- (a) Hardware and Software
- (b) User and Computer Hardware ✅
- (c) RAM and ROM
- (d) Input and Output devices

**Answer: (b)**
> Explanation: OS acts as an intermediary between the user of a computer and the computer hardware.

---

#### Q5. Which statement is TRUE about Operating System?
- (a) OS is optional for computers
- (b) OS is mandatory for computers ✅
- (c) OS is only needed for gaming
- (d) OS is same as application software

**Answer: (b)**
> Explanation: Operating System is mandatory for computers. Without OS, a computer is just a "dead machine" or piece of hardware.

---

### Section B: Kernel and Shell

#### Q6. What is called the "Heart" or "Core" of Operating System?
- (a) Shell
- (b) BIOS
- (c) Kernel ✅
- (d) Bootstrap Loader

**Answer: (c)**
> Explanation: Kernel is called the Heart/Core of Operating System. It is the most essential part of OS.

---

#### Q7. Shell is also known as:
- (a) Core Interpreter
- (b) Command Interpreter ✅
- (c) Hardware Interpreter
- (d) Memory Interpreter

**Answer: (b)**
> Explanation: Shell is called the Command Interpreter. It translates user commands to machine language.

---

#### Q8. Which part of OS loads into RAM during booting?
- (a) Entire Operating System
- (b) Only Shell
- (c) Only Kernel ✅
- (d) Only BIOS

**Answer: (c)**
> Explanation: Only the Kernel part of OS loads into RAM during booting, not the entire Operating System.

---

#### Q9. During shutdown, which program leaves RAM last?
- (a) Application Software
- (b) Shell
- (c) Kernel ✅
- (d) BIOS

**Answer: (c)**
> Explanation: Kernel is the first program to load during startup and the last program to leave RAM during shutdown.

---

#### Q10. Shell contains which of the following?
- (a) Hardware drivers
- (b) Language Translators ✅
- (c) Application programs
- (d) Storage devices

**Answer: (b)**
> Explanation: Shell contains Language Translators like Interpreter, Compiler, and Assembler.

---

#### Q11. Which of the following is a Language Translator found in Shell?
- (a) Interpreter ✅
- (b) Kernel
- (c) BIOS
- (d) Bootstrap

**Answer: (a)**
> Explanation: Interpreter is a language translator found in Shell. Others include Compiler and Assembler.

---

#### Q12. Kernel directly interacts with:
- (a) User
- (b) Application Software
- (c) Hardware ✅
- (d) Shell only

**Answer: (c)**
> Explanation: Kernel directly interacts with hardware and manages system resources.

---

### Section C: Booting Process

#### Q13. What is Booting?
- (a) Shutting down the computer
- (b) Loading OS from Hard Disk to RAM ✅
- (c) Installing new software
- (d) Deleting files

**Answer: (b)**
> Explanation: Booting is the process of loading the Operating System from Hard Disk to RAM.

---

#### Q14. Bootstrap Loader is found in:
- (a) RAM
- (b) Hard Disk
- (c) BIOS (ROM) ✅
- (d) CPU

**Answer: (c)**
> Explanation: Bootstrap Loader is found inside BIOS, which is stored in ROM.

---

#### Q15. What is the full form of BIOS?
- (a) Basic Internal Output System
- (b) Basic Input/Output System ✅
- (c) Binary Input/Output System
- (d) Basic Integrated Operating System

**Answer: (b)**
> Explanation: BIOS stands for Basic Input/Output System.

---

#### Q16. During booting, which component activates first?
- (a) Hard Disk
- (b) RAM
- (c) ROM ✅
- (d) Monitor

**Answer: (c)**
> Explanation: When power is turned on, ROM activates first, which contains BIOS.

---

#### Q17. Bootstrap Loader is also called:
- (a) Kernel
- (b) Shell
- (c) Loader/Boot Loader ✅
- (d) Compiler

**Answer: (c)**
> Explanation: Bootstrap Loader is also called Loader or Boot Loader.

---

#### Q18. Operating System is typically installed in which drive?
- (a) A: Drive
- (b) B: Drive
- (c) C: Drive ✅
- (d) D: Drive

**Answer: (c)**
> Explanation: By default, Operating System is installed in C: Drive.

---

#### Q19. A: and B: drives are reserved for:
- (a) Hard Disk
- (b) Floppy Drives ✅
- (c) CD/DVD
- (d) USB Drives

**Answer: (b)**
> Explanation: A: and B: drive letters are still reserved for Floppy Drives (historical reasons).

---

### Section D: Devices and Embedded Software

#### Q20. Which device does NOT require an Operating System?
- (a) Laptop
- (b) Smartphone
- (c) Washing Machine ✅
- (d) Tablet

**Answer: (c)**
> Explanation: Washing Machine uses Embedded Software, not an Operating System, as it performs only single specific task.

---

#### Q21. Embedded Software is used in:
- (a) Computers
- (b) Smartphones
- (c) Microwave Ovens ✅
- (d) Laptops

**Answer: (c)**
> Explanation: Embedded Software is used in single-purpose devices like Microwave Ovens, Washing Machines, AC, etc.

---

#### Q22. Which is TRUE about Embedded Software?
- (a) Requires regular updates
- (b) No updates required ✅
- (c) Can install new apps
- (d) Used in smartphones

**Answer: (b)**
> Explanation: Embedded Software requires no updates as it is permanently programmed for specific tasks.

---

#### Q23. Calculator does not need OS because:
- (a) It is expensive
- (b) It performs single task only ✅
- (c) It has no screen
- (d) It uses batteries

**Answer: (b)**
> Explanation: Calculator performs only single task (calculations), so it doesn't need an OS.

---

#### Q24. Which device requires an Operating System?
- (a) ATM Machine
- (b) Air Conditioner
- (c) Smartphone ✅
- (d) Microwave

**Answer: (c)**
> Explanation: Smartphone performs multiple tasks, so it requires an Operating System.

---

### Section E: Components of Computer System

#### Q25. How many main components does a computer system have?
- (a) Two
- (b) Three
- (c) Four ✅
- (d) Five

**Answer: (c)**
> Explanation: A computer system has four main components: Hardware, Operating System, Application Software, and Users.

---

#### Q26. Which is the correct order of computer system components (from bottom to top)?
- (a) User → Application → OS → Hardware
- (b) Hardware → OS → Application → User ✅
- (c) OS → Hardware → User → Application
- (d) Application → User → Hardware → OS

**Answer: (b)**
> Explanation: The correct order is: Hardware → Operating System → Application Software → User.

---

#### Q27. Application Software is:
- (a) Mandatory for computers
- (b) Optional - as per requirement ✅
- (c) Part of hardware
- (d) Same as Operating System

**Answer: (b)**
> Explanation: Application Software is optional and installed as per user requirements.

---

#### Q28. Which is NOT an example of Application Software?
- (a) MS Word
- (b) Google Chrome
- (c) Windows 10 ✅
- (d) Adobe Photoshop

**Answer: (c)**
> Explanation: Windows 10 is an Operating System, not Application Software.

---

#### Q29. Users of a computer can be:
- (a) Only humans
- (b) Only machines
- (c) Humans, machines, and robots ✅
- (d) Only programmers

**Answer: (c)**
> Explanation: Users can be people (humans), machines, robots, or other computers.

---

### Section F: OS Functions and Analogy

#### Q30. Operating System is compared to:
- (a) A factory
- (b) A government ✅
- (c) A school
- (d) A hospital

**Answer: (b)**
> Explanation: OS is like a government - it doesn't produce anything but manages and regulates resources.

---

#### Q31. Which is NOT a function of Operating System?
- (a) Resource Allocation
- (b) Priority Management
- (c) Manufacturing products ✅
- (d) Resource Management

**Answer: (c)**
> Explanation: OS doesn't manufacture/produce anything. It only manages and regulates resources.

---

#### Q32. When multiple apps run simultaneously, OS performs:
- (a) Resource Allocation ✅
- (b) Data Entry
- (c) Printing
- (d) Scanning

**Answer: (a)**
> Explanation: When multiple apps run, OS allocates resources (CPU time, memory) to each application.

---

#### Q33. When a phone call comes while playing music, the music stops because:
- (a) Phone is damaged
- (b) Call has higher priority ✅
- (c) Music app crashed
- (d) Battery is low

**Answer: (b)**
> Explanation: Phone call has the highest priority, so OS automatically pauses other tasks.

---

#### Q34. Video song stops when screen is locked because:
- (a) Video needs to be seen and heard ✅
- (b) Video file is corrupted
- (c) Phone is damaged
- (d) Battery saving mode

**Answer: (a)**
> Explanation: Video needs to be both watched and heard. When screen is locked, you can't watch, so OS stops it.

#### Q40. Hardware and Software have:
- (a) No relationship
- (b) One-way relationship
- (c) Mutual relationship ✅
- (d) Competitive relationship

**Answer: (c)**
> Explanation: Hardware and Software have a mutual relationship - both are useless without each other.

---

- (a) Software
- (b) Hardware ✅
- (c) Both at same time
- (d) Neither

**Answer: (b)**
> Explanation: Hardware is always created first, then software is developed for it.

- (a) Body without soul ✅
- (b) Car without wheels
- (c) Book without pages
- (d) Tree without leaves

- (a) Smart machine

### Section I: Mixed/Advanced Questions

#### Q44. Which converts user language to machine language?
- (a) Kernel
- (b) Shell ✅
- (c) CPU
- (d) RAM

**Answer: (b)**
> Explanation: Shell (Command Interpreter) converts any user language to machine language.

---

#### Q45. Kernel understands which language?
- (a) English
- (b) Hindi
- (c) Machine Language ✅
- (d) All languages

**Answer: (c)**
> Explanation: Kernel understands only Machine Language (Binary: 0s and 1s).

---

#### Q46. The process of starting a computer is called:
- (a) Shutting
- (b) Booting ✅
- (c) Loading
- (d) Processing

**Answer: (b)**
> Explanation: The process of starting a computer (loading OS from HDD to RAM) is called Booting.

---

#### Q47. Which is stored in ROM?
- (a) Operating System
- (b) Application Software
- (c) BIOS ✅
- (d) User Data

**Answer: (c)**
> Explanation: BIOS (Basic Input/Output System) is stored in ROM.

---

#### Q48. OS provides environment for:
- (a) Hardware to work
- (b) Application programs to run ✅
- (c) Users to sleep
- (d) Electricity to flow

**Answer: (b)**
> Explanation: OS provides an environment for application programs to run.

---

#### Q49. Which manages computer hardware?
- (a) User
- (b) Application Software
- (c) Operating System ✅
- (d) Printer

**Answer: (c)**
> Explanation: Operating System manages computer hardware and provides interface for applications.

---

#### Q50. Startup of computer is also called:
- (a) Shutdown
- (b) Restart
- (c) Booting ✅
- (d) Sleeping

## Important Points to Remember for Exam

1. **OS = System Software**
2. **Kernel = Heart/Core of OS**
3. **Shell = Command Interpreter**
4. **Booting = Loading OS from HDD to RAM**
5. **Only Kernel loads to RAM during booting**
6. **Kernel is last to leave RAM during shutdown**
7. **A: and B: drives = Reserved for Floppy**
8. **C: drive = Default for OS installation**
9. **BIOS = Basic Input/Output System (in ROM)**
10. **Embedded Software = For single-purpose devices, no updates needed**
